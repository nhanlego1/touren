Create a WordPress Options Page out of the box. Specify the options (images or text) and use them even inside posts using short-codes

== Description ==

Create Options Pages in less than 2 minutes! Specify the **Title** of the page, the **icon**, and the **options**.

1. Select the icon for your page from a list of icons, all of them prepared and optimized for > 3.8 wordpress admin panel.

2. For the options you can specify text string or image. (more options are coming in next updates!)

3. We also provide \"short-codes\" to use the options inside your posts.

Do you use WordPress admin with an iPad? Don´t worry, this plugin is fully adapted for tablets and smartphones.

*Thank you for downloading! your feedback is well appreciated!*

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload plugin folder `easy-options-page` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= The Image is not saved correctly =

Check the permissions of the "uploads" folder.

= I cannot see the Options page in the admin menu =

Reload the page, or clean your browser cache.
